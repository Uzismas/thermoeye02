# ThermoEye Project

A clinical console application for thermal imaging analysis and patient management.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app` - Next.js app directory
- `/features` - Feature modules
- `/scripts` - Utility scripts
- `/public` - Static assets
